# Meco Studio

Meco Studio 是一个 **OpenClew（OpenClaw）智能体管理项目**，主打更快、更安全的本地化智能体协作，支持多智能体圆桌讨论与实时音频推流。

## 一行安装（Install In One Command）

```bash
curl -fsSL https://raw.githubusercontent.com/EdenShadow/mecostudio/main/scripts/install-meco-studio.sh | bash
```

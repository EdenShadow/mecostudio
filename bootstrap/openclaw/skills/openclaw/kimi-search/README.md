# 🔍 Kimi Search - OpenClaw Skill

使用 **Kimi AI** 进行联网搜索，自动整理搜索结果并构建本地知识库。

## ✨ 核心特性

- 🌐 **联网搜索** - 使用 Kimi AI 实时搜索互联网
- 📚 **知识库构建** - 自动整理并保存搜索结果
- 📑 **智能索引** - 自动生成知识库索引
- 📖 **快捷指南** - 自动生成使用说明书
- 🗂️ **分类管理** - agents/tech/research/reference 分类
- ⚡ **多种深度** - quick/standard/deep 三种搜索模式

## 🚀 快速开始

```bash
# 基础搜索
kimi-search "搜索内容"

# 指定分类搜索
kimi-search "赛博朋克AI助手性格特点" --category agents

# 深度搜索
kimi-search "React Server Components 最佳实践" --depth deep

# 更新索引
kimi-search index
```

## 📦 安装

此 skill 已安装在 OpenClaw 中：

```bash
openclaw skills list | grep kimi-search
```

如需手动安装：
```bash
cd ~/.openclaw/skills/kimi-search
npm link
```

## 📚 使用场景

### 1. 创建新Agent时查找人设
```bash
kimi-search "赛博朋克风格AI助手的性格特点和说话方式" \
  --category agents \
  --context "用于RPG游戏"
```

### 2. 寻找技术知识库
```bash
kimi-search "MCP 协议规范和实现" --category tech --depth deep
```

### 3. 快速事实查证
```bash
kimi-search "OpenAI o3-mini 模型特点" --depth quick
```

### 4. 构建领域知识库
```bash
kimi-search "2024年AI行业发展趋势" --category research --depth deep
```

## 🛠️ 命令参考

| 命令 | 说明 |
|------|------|
| `kimi-search <query>` | 执行搜索 |
| `--category, -c <name>` | 指定分类 (agents/tech/research/reference/general) |
| `--depth, -d <level>` | 搜索深度 (quick/standard/deep) |
| `--context, -x <text>` | 添加上下文背景 |
| `kimi-search index` | 更新知识库索引 |
| `kimi-search guide` | 生成快捷指南 |
| `kimi-search list` | 列出知识库内容 |
| `kimi-search --help` | 显示帮助 |

## 🔧 配置

### 环境变量
```bash
# 自定义知识库位置
export KIMI_SEARCH_DIR=/path/to/knowledge-base
```

### 依赖
- **kimi-cli** - 必须安装并配置API Key
- **Node.js** >= 18.0.0

安装 kimi-cli：
```bash
curl -LsSf https://code.kimi.com/install.sh | bash
```

配置 API Key：
```bash
kimi
# 输入 /login 配置 Kimi API Key
```

## 💡 搜索技巧

1. **使用具体关键词** - 越具体结果越准确
2. **添加上下文** - 使用`--context`说明用途
3. **选择合适深度**：
   - `quick` - 快速查证 (~500字)
   - `standard` - 标准搜索 (~1000-1500字)
   - `deep` - 深度研究 (~2000-3000字)
4. **定期整理** - 使用 `kimi-search index` 更新索引

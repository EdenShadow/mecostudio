## 📊 分类说明

### 🚀 科技趋势
- AI、人工智能、机器学习
- 编程、代码、开发
- 软件、硬件、科技产品
- **关键词**: 科技、AI、技术、软件、编程、tech、software

### 📈 财经投资
- 股票、美股、A股
- 加密货币、比特币、区块链
- 投资理财、基金
- **关键词**: 财经、投资、股票、加密货币、finance、invest

### 👥 人物推荐
- X 博主推荐、账号推荐
- KOL、专家、行业领袖
- **关键词**: 博主、账号、关注、人

### 🛠️ 工具资源
- 效率工具、软件推荐
- APP、网站、资源
- **关键词**: 工具、软件、APP、网站、资源

### 📚 学习成长
- 教程、课程、学习资源
- 技能提升、知识分享
- **关键词**: 学习、教程、课程、技能、知识

### 🌟 生活百科
- 生活技巧、健康养生
- 美食、旅行、生活方式
- **关键词**: 生活、健康、美食、旅行

## 📄 文件命名规则

```
[Timestamp]_[问题摘要].txt
Example: 20260211_143000_帮我找知名科技博主.txt
```

## 📋 账号列表

对于人物推荐类查询，会额外生成账号列表文件：

```
[Timestamp]_账号列表.txt
Example: 20260211_143000_账号列表.txt
```

## 🎯 使用方法

```bash
# 运行查询
~/.openclaw/skills/x-grok/xgrok "帮我找一些x上知名的科技博主"

# 结果会自动保存到对应的分类文件夹
~/Documents/知识库/我的助手/科技趋势/20260211_143000_帮我找知名科技博主.txt
```

## 💡 提示

- 查询会自动根据关键词分类
- 每个文件都包含高质量整理的结果
- 账号和链接会被提取并整理
- 所有结果都带有时间戳便于追溯
